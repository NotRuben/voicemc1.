VoiceMC Towny Ranks

VoiceMC Towny 2.5
New Features:
-New hunting quest line.
-Added “Cleanse Charm” when used on an item it will remove any curse enchantments.
-Added “Rage Trinket” when you have 2 or less hearts deal 60% more damage.
-Added “Mob Essence” when killing certain hostile mobs it will be dropped.
-Added “Voice Items” artifact category.
-New category contains 5 versions of the “Voice Sword”
-New custom enchants “Viper I, LifeSteal I, and DoubleDamage I” have all been added.


Bug Fixes:
-Scutes now correctly drop from the iron crate.
-Afk Fishing plugin will no longer kick you for afk fishing.
-/feed cooldown is now matched with /heal at 5 minutes.
-Numerous shop bugs have been fixed.
-Tridents can no longer be thrown in towns.
-Sleeping plugin will no longer broadcast.
-Melons will now work with the farmer job.
-Attempted a McMMO potion fix.

VoiceMC Towny 2.4
New Features:
-SekShop
-Prices will now update depending on the supply of the item.
-Each item has a max price, min price, and supply limit.
-Added U L T R A rank.
-Diamond crate now has a chance to drop a Voice Crate Key.
-Phantom Slayer now has Impaling III, Loyalty III, and Channeling I.

Bug Fixes:
-Fixed players obtaining a negative balance.
-Fixed quests not dropping artifacts correctly.
-Fixed starting a quest twice.
-Other bug fixes.

VoiceMC Towny 2.3 and 2.2

-Fixed Beta Quest NPC reseting the quest.
